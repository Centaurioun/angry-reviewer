Angry Reviewer
Corrector Rules About

The Angry Reviewer is a free and open-source project. These algorithms were gathered and implemented by Roman Anufriev based on articles, books, and lectures on scientific writing as well as his experience in academia.

Privacy

This website does not save or store your text, as you can verify in the source code. If you still worry about privacy of your super secret project, you can download the python code of this algorithm with the link below, verify it, and run it locally on your computer.

Plugins

Love Vim? So do I, and here is a Vim plugin with offline AngryReviewer. Plugins for other platforms are in development. If you know how to develop plugins for Word, LibreOffice, or web browsers, your help would be highly appriciated!
